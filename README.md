## Introduction:

This application is a whatsapp clone using React + firebase

## Start:

```sh
$ git clone https://github.com/rishi-gaikwad/whatsapp-clone
```

```sh
$ cd whatsapp-clone
```

```sh
$ yarn install
```

```
$ yarn start
```
